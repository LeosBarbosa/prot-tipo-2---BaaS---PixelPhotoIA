🔹 Prompt: Geração de Logotipo Minimalista
🧠 Função:
Crie um logotipo vetorial minimalista e de alto contraste, utilizando o conceito de um ícone de montanha para uma marca de aventura. O resultado deve ser limpo e otimizado para identidade visual.
⚙️ Instrução para GEM Build:
Chame a função de serviço generateLogo(prompt: string), usando a descrição do ícone como parâmetro de entrada. O modelo deve gerar uma imagem no formato 1:1, aplicando automaticamente a diretriz interna de ser um "logotipo vetorial minimalista" com linhas limpas e cores chapadas.
Exemplo de chamada:
geminiService.generateLogo(
  "ícone minimalista de montanha para uma marca de aventura, com design geométrico e cores planas (preto e branco)"
);

Parâmetros da Ferramenta:
 * prompt: "ícone minimalista de montanha para uma marca de aventura"
 * output_style: "Minimalist vector logo, Simple, clean lines, flat colors, high contrast, suitable for a brand. Centered on a white background."
 * output_aspect_ratio: "1:1"
🔧 Ferramenta Destinada:
logoGen (Gerador de Logotipo AI)
